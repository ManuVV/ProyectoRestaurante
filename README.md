🍽️ Proyecto Restaurante  

Aplicación web completa para la gestión de un restaurante desarrollada con Angular y Spring Boot.  

El proyecto permite a los usuarios consultar la carta digital, registrarse, iniciar sesión y realizar reservas, mientras que los administradores pueden gestionar platos, usuarios y reservas desde un panel de administración.  

🚀 Características  
👤 Usuarios  
· Registro e inicio de sesión  
· Autenticación mediante JWT  
· Visualización de la carta digital  
· Realización de reservas  
· Confirmación de reservas mediante correo electrónico
